## API

| Prop Name  | Type     | Default  | Description                                               |
| ---------- | -------- | -------- | --------------------------------------------------------- |
| `text`     | `string` | Required | The text to be displayed with the hover effect.           |
| `duration` | `number` | `null`   | The duration of the mask transition animation in seconds. |
